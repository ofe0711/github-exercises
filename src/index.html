Github Exercises

PartI

  1. Create a local repository and add and commit some files
    • mkdir git_practice_exercise
    • cd git_practice_exercise
    • touch quick_learning.txt
    • git init
    • git add .
    • git commit -m "Adding quick_learning.txt"
  2. Create a remote repository and push your code from the local repo to the remote
    • open your page on github
    • click create repository
    • give repository a name
    • go to terminal, git remote add origin "link of repository copied from clipboard"
    • git push origin master
  3. Go to this link and fork. https://github.com/rithmschool/git_practice
    • go to terminal and clone. i.e git clone "link copied from clipboard"
  4. Push origin clone master
  5. submit pull request
